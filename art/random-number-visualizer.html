Large Number Visualizer Using Pixels

The top number is the number of (random) dots being drawn. The bottom number is the speed the dots are being drawn.

This isn't fully accurate, since I do not check if there is already a dot in the location before I draw it. So, when drawing higher numbers of dots it iss almost certainly re-drawing over itself. Still, it gives you a perspective of the time it takes to draw that many dots.

Controls

Up/Click
-
10x the number of dots
Down
-
1/10 the number of dots
Right
-
10x the speed of the dots
Left
-
1/10 the speed of the dots
